# 🌸 WaifuDex

WaifuDex is a simple web application inspired by the concept of a Pokédex, but focused on anime characters.

The project was created as part of a school project using only HTML, CSS and JavaScript.

---

## ✨ Features

- 🔎 Search anime characters
- 🖼️ Character images
- 📖 Character information
- 📱 Responsive interface

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
